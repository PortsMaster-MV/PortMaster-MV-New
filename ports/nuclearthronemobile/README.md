## Controls

Note:  When stuck on achievement screen after end of a run, use select or L3 buttons.

|Button| Action |
|--|--|
| l-stick| movement  |
| r-stick| aim |
| R1| primary attack |
| L1| ability |
| A| select/pick up |
| B| back/cancel |
| R2| change weapon |
| Start| pause |
