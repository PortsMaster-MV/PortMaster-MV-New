## Controls

| Button | Action |
|--|--| 
|dpad|movement|
|left-analog|movement|
|Start|Pause/Resume|
|A button|Holy Power|
|B button|Jump|
|Y button |Attack/Weapon|
|Y + Up Combo|Use Sub Weapon |
|X button|Switch Sub Weapons|
|L1 shoulder|Backwards Dash|
|R1 shoulder |Block|
