## Controls

| Button | Action |
|--|--| 
|DPAD / Left stick|Movement|
|A|Interact / Confirm (Menu)|
|B|Move Faster / Go Back (Menu)|
|X|Pick Item from Inventory|
|Y|Change Interaction Options|
|R1|Quick Snapshot|
|Start|Pause Game|
|Select|Open Chao Menu|
